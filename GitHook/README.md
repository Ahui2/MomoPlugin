# Github WebHook监听器
- 在配置文件中配置正确的Hook地址即可监听git仓库

## 指令
| 名称                                     |  说明           |    权限    |
|----------|:------------------:|:------------------:|
| /git     |   管理监听事件  |     无           |


## 配置
> GitWebHook
```json
{
  "启用": true,
  "路由": "/update/",
  "端口": 7000,
  "监听事件": {
    "release": [
      816771079
    ],
    "star": [
      816771079
    ]
  }
}
```
